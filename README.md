# 재태크 팀 게임 — 프로토타입

React + Vite + Tailwind + Zustand + Recharts

## 실행 방법
```bash
npm install
npm run dev
```
http://localhost:5173 접속

## 주요 기능
- 팀 생성, 팀원 관리, 시드머니 자동 지급
- 2000년 → 2015년, 총 15라운드
- 라운드 시작 시 사건 힌트(무작위) 표시
- 업종/종목별 분산 투자(퍼센트 슬라이더)
- 라운드 종료 시 사건 효과를 수익률로 반영 후 자산 업데이트
- 라운드별 자산 변동 라인 차트
- 리더보드, 최종 결과(우승팀/라운드별 기록)
